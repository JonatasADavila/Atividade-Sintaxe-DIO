# 📋 Atividade Sintaxe DIO

Aplicação Java de console com menu interativo desenvolvida como atividade do bootcamp da DIO. O programa apresenta um menu com operações matemáticas básicas, permitindo ao usuário escolher entre diferentes cálculos de forma simples e intuitiva.

---

## 🗂️ Funcionalidades

O sistema possui um menu com as seguintes opções:

**[1] Calculadora de Idade**
Solicita o nome do usuário e o ano em que nasceu, e calcula automaticamente a sua idade com base no ano atual.

**[2] Área do Quadrado**
Solicita o valor do lado de um quadrado e retorna a sua área.

**[3] Área do Triângulo**
Solicita a base e a altura de um triângulo e retorna a sua área.

**[4] Diferença de Idade entre Duas Pessoas**
Solicita o nome e a idade de duas pessoas e calcula a diferença de idade entre elas.

**[0] Sair**
Encerra o programa.

---

## 🛠️ Tecnologias Utilizadas

- **Java 25 no Intellij**

## 📚 Contexto

Este projeto foi desenvolvido como parte de uma atividade prática do bootcamp da [DIO (Digital Innovation One)](https://www.dio.me/), com o objetivo de praticar conceitos básicos de sintaxe Java, como:

- Entrada de dados com `Scanner`
- Estruturas condicionais (`switch`)
- Operações aritméticas
- Estrutura de repetição com menu interativo
